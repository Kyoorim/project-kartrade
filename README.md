## 🥑프로젝트 소개

- next.js와 typescript로 구현한 포토카드 거래 사이트입니다. 
- 헤딩 포토카드를 클릭하면 상세 페이지로 이동할 수 있습니다.
- 검색기능 및 가격순 정렬을 설정할 수 있고 동적라우팅을 구현했습니다. 
- 반응형으로 구현했습니다. 
- figma로 미리 작업되어있는 디자인을 그대로 구현했습니다.
